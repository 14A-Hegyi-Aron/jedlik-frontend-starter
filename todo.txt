1. GitHub repo klónozása, átnevezése
    https://github.com/nitslaszlo/jedlik-frontend-starter
2. Források, backend szerver másolása, backend szerver indítása
3. Sablon megnyitása, Quasar csomagverziók frissítése (ha kéri a feladat), Node.js csomagok telepítése
4. Axios backend szerver beállítása - Base URL és port: src/store/axios.instance.ts
5. "1"-oldali store IFields interface és axios URL-ek aktualizálása: src/store/store1.ts
6. "N"-oldali store IFields interface, URL-ek és actions metódusok aktualizálása: src/store/storeN.ts
7. Nyitó oldal elkészítése: src/pages/index.vue
8. Komponensek (oldalak) elkészítése: src/pages mappában ("save as" indulással)
9. Route-ok elkészítése az új oldalakhoz: src/routes.ts 
10. "Takarítás-törlés": oldalak, menük, route-ok, állapotsor ki-be kapcsolása, stb.:
   - src/App.vue,
   - src/routes.ts,
   - src/store/appStore.ts,
   - src/pages/x*.vue állományok törlése


Nyomkövetés:
1. Indítsad el a fordítást (Ctrl-Shift-B, vagy npm run dev)
2. Zárjad be a megjelenő böngészőablakot
3. Helyezd el a töréspontot/töréspontokat a forrásban
4. F5 -el indítsad el a nyomkövetést
5. A megjelenő böngészőablakban "tevékenykedj" úgy, hogy a breakpoint-ra kerüljön a vezérlés
6. Vizsgáld meg a változók aktuális értékeit, figyeld meg a végrehajtásra kerülő utasításokat (F10, F11)
7. Folytasd a nyomkövetést: F5, F10, F11, stb. (lsd. VS Code - Run menüben a funkciókat)
