1. GitHub repo klónozása, átnevezése
    https://github.com/nitslaszlo/jedlik-frontend-starter
2. Források, backend szerver másolása, backend szerver indítása, ha a szerver lokális gépen fut
3. Sablon megnyitása, Quasar csomagverziók frissítése (ha kéri a feladat), Node.js csomagok telepítése
4. Axios backend szerver beállítása - Base URL és port(ha adott): src/store/axios.instance.ts
5. "1"-oldali "IOne" interface és axios URL-ek, akciómetótusok aktualizálása: src/store/store.ts
6. "N"-oldali "IMany" interface, URL-ek és akciómetódusok aktualizálása: src/store/store.ts
7. Egyéb táblához (ha van) "IOther" interface, URL-ek és akciómetódusok készítése: src/store/store.ts
8. Nyitó oldal elkészítése (források beillesztése): src/pages/index.vue
9. Komponensek (oldalak) elkészítése (források beillesztése): src/pages mappában ("save as" indulással)
10. Route-ok elkészítése az új oldalakhoz: src/routes.ts 
11. "Takarítás-törlés": oldalak, menük, route-ok, állapotsor ki-be kapcsolása, stb.:
   - src/App.vue,
   - src/routes.ts,
   - src/store/store.ts,
   - src/pages/x*.vue állományok törlése


Nyomkövetés:
1. Indítsad el a fordítást (Ctrl-Shift-B, vagy npm run dev)
2. Zárjad be a megjelenő böngészőablakot
3. Helyezd el a töréspontot/töréspontokat a forrásban
4. F5 -el indítsad el a nyomkövetést
5. A megjelenő böngészőablakban "tevékenykedj" úgy, hogy a breakpoint-ra kerüljön a vezérlés
6. Vizsgáld meg a változók aktuális értékeit, figyeld meg a végrehajtásra kerülő utasításokat (F10, F11)
7. Folytasd a nyomkövetést: F5, F10, F11, stb. (lsd. VS Code - Run menüben a funkciókat)
